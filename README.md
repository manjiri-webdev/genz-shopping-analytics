# Gen-Z Shopping Analytics Dashboard

An interactive dashboard built using R Shiny to analyze Gen-Z online shopping behavior based on survey data.

## Features
- Data visualization
- Correlation & regression analysis
- Poisson distribution
- ANOVA & segmentation

## Tech Stack
- R
- Shiny
- ggplot2
- dplyr
